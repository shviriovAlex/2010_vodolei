Как подключить электричество

Памятка по прокладке кабеля

Для подключения электроснабжения по постоянной схеме необходимо:

1. Погасить задолженность (если имеется) по взносам и пене, произвести, при необходимости, покос травы (участок должен быть ухожен).

2. Написать заявление в правление товарищества на получение технических условий, выслать его на электронный адрес stvodoley2010@gmail.com, и указать свой электронный адрес, на который Вам будут высланы технические условия.

3. После выполнения технических условий позвонить электрику товарищества для согласования даты подключения электроснабжения.

4. При подключении электроснабжения иметь при себе:

- членскую книжку садовода;

- квитанцию об оплате 50 кВт электроэнергии на счет товарищества (оплата через ЕРИП);

- заявление на подключение к электроснабжению (только в том случае, если заявление на электронный адрес направлено без подписи члена товарищества);

- два экземпляра технических условий (1 экземпляр после подписания электриком товарищества остается у владельца участка, второй экземпляр в правлении товарищества).

Для подключения электроснабжения по временной схеме необходимо:

Установить розетку внутри электрического ящика, соответствующего номеру Вашего участка, не срывая пломбы с вводного автомата и прибора учета электроэнергии. Данную работу может выполнить электрик собственника участка.
